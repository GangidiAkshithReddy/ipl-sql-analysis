# IPL Cricket Data Analysis using SQL

## Project Overview
Exploratory data analysis on IPL cricket dataset using PostgreSQL.
Dataset contains 1095 matches and 260,920 deliveries from 2008 to 2022.

## Tools Used
- PostgreSQL
- pgAdmin

## Dataset
- matches.csv — match level data
- deliveries.csv — ball by ball data

## Business Questions Answered
1. How many matches were played in each season?
2. Which team won the most matches overall? → Mumbai Indians (144 wins)
3. Which city hosted the most matches? → Mumbai (235 matches)
4. Which player won the most Player of the Match awards? → AB de Villiers (25 awards)
5. How many matches did each team win by runs vs by wickets?
6. Which venue had the most matches? → Eden Gardens (77 matches)
7. Who are the top 10 batsmen by total runs? → V Kohli leads with 8014 runs
8. Who are the top 10 bowlers by total wickets? → YS Chahal leads with 213 wickets
9. Which team scored the most total runs? → Mumbai Indians (42176 runs)
10. What is the average runs scored per match in each season?

## Key Insights
- Mumbai Indians dominated IPL with most wins and highest total runs
- V Kohli is the highest run scorer in IPL history
- YS Chahal is the highest wicket taker in IPL history
- Mumbai city hosted the most matches across all seasons
